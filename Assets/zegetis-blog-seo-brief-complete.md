## **Global SEO Requirements (Apply to All Posts)**

### **Technical SEO**
- **URL Structure:** `zegetis.be/blog/[post-slug]`
- **Canonical:** Self-referencing on each post
- **Sitemap:** Include in `/sitemap.xml`, priority 0.7
- **Mobile-first:** Responsive, fast load (<2.5s LCP)
- **Images:** WebP format, alt text with target keyword
- **Internal Linking:** Min 3 internal links per post

### **Schema Markup (JSON-LD) — BlogPosting**
```json
{
  "@context": "https://schema.org",
  "@type": "BlogPosting",
  "headline": "[Post Title]",
  "description": "[Meta Description]",
  "image": "[Featured Image URL]",
  "datePublished": "[YYYY-MM-DD]",
  "author": {
    "@type": "Person",
    "name": "Daan Hannon"
  },
  "publisher": {
    "@type": "Organization",
    "name": "Zeget'is"
  }
}
```
